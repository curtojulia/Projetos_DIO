Primeira Stack com AWS CloudFormation
- Repositório criado como parte do desafio da DIO sobre AWS CloudFormation.

O que foi feito:
O arquivo de template disponibilizado pela DIO foi dividido em 4 partes, cada uma representando uma stack separada. A partir disso, foi criada e testada a primeira stack na AWS utilizando o arquivo 01-EC2.yaml, que provisiona uma instância EC2 simples.

O que é AWS CloudFormation
É um serviço da AWS que permite criar e gerenciar recursos de infraestrutura através de código, eliminando a necessidade de configuração manual pelo console.

Tecnologias
AWS CloudFormation
Amazon EC2
