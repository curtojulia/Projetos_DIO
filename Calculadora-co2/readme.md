# Calculadora de Emissão de CO₂ por Meio de Transporte

Este projeto é uma aplicação **estática** em **HTML, CSS e JavaScript** que calcula a quantidade estimada de emissão de CO₂ de acordo com o meio de transporte escolhido e a distância percorrida.

A calculadora permite:
- Informar a **distância** percorrida;
- Escolher o **meio de transporte** (carro, moto, ônibus, etc.);
- Estimar a **emissão de CO₂** gerada;
- Comparar diferentes meios de transporte;
- Exibir sugestões de redução de impacto ambiental e créditos de carbono.

## Base do projeto

Este projeto foi desenvolvido seguindo as aulas de um curso, utilizando:
- **Os mesmos dados e lógica de cálculo do professor** para as emissões de CO₂;
- A mesma estrutura de funcionalidade (inputs, cálculo, comparação, dicas).

A principal diferença é que eu:
- **Personalizei o front-end**, alterando a paleta de cores para um tema em **azul pastel**;
- Ajustei o layout e alguns detalhes visuais para deixar a interface mais agradável e moderna, mantendo a mesma base lógica do curso.

## Tecnologias utilizadas

- **HTML5**  
- **CSS3** (com uso de variáveis CSS e responsividade)  
- **JavaScript** (cálculos, manipulação do DOM e exibição dos resultados)
